WinlogonMessageBlocker   v.1.0.0.2

此工具实现对 x86 和 x64 系统的 winlogon 消息的完全拦截。

目前此工具支持 win7 及以上操作系统，程序有 GUI 界面和自动系统版本识别。


使用方法：完全解压 WinlogonMessageBlocker.zip 压缩文件，然后运行
 StartupManager.exe 程序并按照程序提示操作。
